# Browser Privacy & Evidence Leakage Analyzer

## Description
This cyber forensic tool analyzes browser artifacts to identify privacy leaks
and residual digital evidence left behind by users.

## Forensic Relevance
- Browser history analysis
- Privacy leakage detection
- Digital evidence reconstruction

## Features
- Extracts recent browsing history
- Detects sensitive browsing patterns
- Read-only forensic approach

## Requirements
- Python 3.x
- Google Chrome (Chromium based)

## Usage
1. Run the script:
   python browser_analyzer.py
2. View extracted forensic artifacts
